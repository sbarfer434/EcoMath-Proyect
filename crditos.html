Descripción

Este recurso está dirigido a 1º ESO, en él se pretende concienciar al alumnado sobre la necesidad de realizar un consumo energético responsable. Para ello se pedirá a los participantes un estudio del consumo energético del centro, propuestas para disminuir ese consumo y acciones para concienciar a los compañeros y compañeras. Finalmente se pedirá que se diseñe la reforma de las aulas menos eficientes del centro, realizando un presupuesto que incluya precio del material: pintura, ventanas, puertas, así como paneles acústicos para reducir también la contaminación acústica de las aulas, así como la realización de una maqueta a escala de la propuesta de reforma.

Para realizar esta tarea es necesario utilizar elementos matemáticos por lo que se muestran estos a lo largo del recurso. En concreto se trabajan la geometría de polígonos y la proporcionalidad. El dominio de estos conceptos ayudará a realizar el proyecto para el cambio mediante la creación de maquetas con las propuestas.

Este proyecto se ha desarrollado siguiendo los principios metodológicos de AICLE (Aprendizaje Integrado de Contenidos y Lenguas Extranjeras) y DUA (Diseño Universal para el Aprendizaje), garantizando así la inclusión y la atención a la diversidad del alumnado, además de fomentar el aprendizaje de contenidos de manera integrada en una lengua extranjera.
