Saving Energy in Our Classroom / Ahorrando energía en nuestro aula

Easy Reading Dictionary / Diccionario lectura facilitada

Efficient Consumption / Consumo eficiente

Definition / Definición

Using energy wisely to reduce waste. / Usar la energía de manera inteligente para reducir el desperdicio.

Example / Ejemplo

We need to improve efficient consumption at school. / Necesitamos mejorar el consumo eficiente en la escuela.

Renewable Materials / Materiales renovables

Definition / Definición

Materials that come from natural sources and can be replaced. / Materiales que provienen de fuentes naturales y pueden reponerse.

Example / Ejemplo

We should use renewable materials in our classroom renovation. / Deberíamos usar materiales renovables en la renovación del aula.

Insulation / Aislamiento

Definition / Definición

Material that helps to keep heat inside or outside. / Material que ayuda a mantener el calor dentro o fuera.

Example / Ejemplo

Good insulation reduces energy use. / Un buen aislamiento reduce el uso de energía.

Energy Efficiency / Eficiencia energética

Definition / Definición

Using less energy to do the same work. / Usar menos energía para hacer el mismo trabajo.

Example / Ejemplo

LED lights improve energy efficiency. / Las luces LED mejoran la eficiencia energética.

Natural Ventilation / Ventilación natural

Definition / Definición

Using windows and airflow to keep spaces fresh. / Usar ventanas y el flujo de aire para mantener los espacios frescos.

Example / Ejemplo

Natural ventilation reduces the need for air conditioning. / La ventilación natural reduce la necesidad de aire acondicionado.

Sustainable Design / Diseño sostenible

Definition / Definición

Planning spaces to be eco-friendly and efficient. / Planificar espacios para que sean ecológicos y eficientes.

Example / Ejemplo

We are learning about sustainable design for our school. / Estamos aprendiendo sobre diseño sostenible para nuestra escuela.

Rhombus / Rombo

Definition / Definición

A shape with four equal sides, but its angles are not 90 degrees. / Una figura con cuatro lados iguales, pero sus ángulos no son de 90 grados.

Example / Ejemplo

The diamond shape is a rhombus. / La forma del diamante es un rombo.

Diagonals / Diagonales

Definition / Definición

Lines that go from one corner to the opposite corner of a shape. / Líneas que van de una esquina a la esquina opuesta de una figura.

Example / Ejemplo

The diagonals of a square are equal in length. / Las diagonales de un cuadrado son iguales en longitud.

Trapezoid / Trapecio

Definition / Definición

A shape with four sides, where one pair of sides is parallel. / Una figura con cuatro lados, donde un par de lados es paralelo.

Example / Ejemplo

The roof of the house is shaped like a trapezoid. / El techo de la casa tiene forma de trapecio.

Rhombus / Rombo

Definition / Definición

A shape with four equal sides, but its angles are not 90 degrees. / Una figura con cuatro lados iguales, pero sus ángulos no son de 90 grados.

Example / Ejemplo

The diamond shape is a rhombus. / La forma del diamante es un rombo.

Perimeter / Perímetro

Definition / Definición

The total distance around a shape. / La distancia total alrededor de una figura.

Example / Ejemplo

The perimeter of a square is four times its side length. / El perímetro de un cuadrado es cuatro veces la longitud de su lado.

Area / Área

Definition / Definición

The space inside a shape. / El espacio dentro de una figura.

Example / Ejemplo

The area of a rectangle is its length times its width. / El área de un rectángulo es su longitud por su ancho.

Rectangle / Rectángulo

Definition / Definición

A shape with four sides and four right angles. / Una figura con cuatro lados y cuatro ángulos rectos.

Example / Ejemplo

A door is usually in the shape of a rectangle. / Una puerta suele tener forma de rectángulo.
